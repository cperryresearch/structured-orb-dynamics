# Geometry-First Motion Analysis (Methodology)

This directory contains methodological companion documents describing a
geometry-first, descriptive approach to motion analysis under data constraints.

The work here is concerned with **how motion is approached**, not with
explaining, predicting, or inferring causes beyond what the data can support.

The focus is on:
- observable geometric structure
- explicit assumptions and scope
- reproducible, falsifiable analytical procedures

These documents are intended to **sit alongside** application-specific analyses
(including *Structured Orb Dynamics*), not to replace or compete with them.
